## Idee directrice

Le but n'est pas de memoriser Python comme un dictionnaire.
Le but est de voir le systeme:

- ce que tu stockes
- ce que tu compares
- ce que tu transformes
- ce que tu retournes

## Regle ENZI

Quand tu bloques, reviens toujours a cette carte:

```text
donnees -> operateurs -> conditions -> boucles -> fonctions -> structures
```

Les operateurs sont le moteur local.
Les structures de controle sont la transmission.
Les fonctions et classes sont le chassis.
