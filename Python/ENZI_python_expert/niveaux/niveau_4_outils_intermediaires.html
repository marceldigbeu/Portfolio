(System) Tu sors du Python scolaire pur pour entrer dans le Python qui va plus vite et s'organise mieux.

[Critical Node] Choisir le bon outil standard avant d'ecrire une boucle inutile.

Blocs a maitriser

  • list, dict et set comprehensions
  • sum, max, min, sorted, enumerate, zip
  • modules utiles: math, random, pathlib
  • lecture et ecriture de fichiers

[Mechanics]

notes = [12, 8, 19, 14]
admis = [note for note in notes if note >= 10]
total = sum(admis)

print(admis)
print(total)

La comprehension est une chaine de montage compacte: elle lit, filtre et reconstruit en une seule passe lisible.

Operateurs qui reviennent

  • for ... in ... if ... dans les comprehensions
  • | pour fusionner des dictionnaires modernes ou faire une union de sets
  • with pour encapsuler un fichier proprement

Mission

1. Construis une liste des notes admises avec une comprehension
2. Trie cette liste avec sorted()
3. Ecris le resultat dans un fichier texte
4. Relis le fichier et affiche son contenu
